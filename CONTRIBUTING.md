# Contributing

Thanks for helping with Codex Tomogatchi.

## Development Setup

Install Python 3, Node.js LTS, and npm.

Windows:

```powershell
.\scripts\setup.ps1 -SkipMarketplace
```

macOS/Linux:

```bash
./scripts/setup.sh --skip-marketplace
```

## Checks

Run:

```powershell
npm test
py -3 scripts/validate_assets.py
```

The CI workflow runs:

- Python unit tests.
- Electron JavaScript syntax checks.
- Pet asset validation.
- Codex plugin manifest validation when the validator is available.

## Privacy Requirements

Changes must preserve the privacy contract:

- Do not store prompt text.
- Do not store command text.
- Do not store tool output.
- Do not store raw hook payloads or raw session-log records.

Add or update tests whenever a change touches session-log ingestion, reactions, settings, lifecycle, care calls, or evolution rules.

## Assets

Bundled default assets must be original or explicitly licensed for redistribution. The current default sprites are generated by:

```powershell
py -3 scripts/generate_original_assets.py
```

Do not commit proprietary or fandom-inspired art as bundled defaults.
